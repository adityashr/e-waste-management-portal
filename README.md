# E-Waste Management Portal

## Overview
The E-Waste Management Portal is a web-based platform designed to streamline the process of electronic waste collection, recycling, and management. It connects users, recyclers, and administrators to ensure responsible e-waste disposal and promote environmental sustainability.

## Features
- **User Registration & Login:** Secure accounts for users to request pickups.
- **Request E-Waste Pickup:** Schedule collection of old electronics from home or office.
- **Admin Dashboard:** Manage users, pickups, and track statuses.
- **Recycler Integration:** Connects with certified e-waste recyclers for proper disposal.
- **Status Tracking:** Users can track the progress of their e-waste pickups.
- **Secure & Responsive:** Works on desktop and mobile devices with a user-friendly interface.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Authentication:** Session-based login system
